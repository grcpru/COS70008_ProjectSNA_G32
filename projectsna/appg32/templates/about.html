Project Information

Project Title:

Design and Implementation of a Cross-platform Desktop Application for Social Network Analysis using Statistical Models

Project Description:

The goal of this project is to develop a desktop application for social network analysis using statistical models, including models such as exponential random graph models (ERGMs) or auto-logistic actor attribute models (ALAAMs). The application will provide users with a user-friendly and interactive interface to collect and upload network data, perform network descriptive statistics, visualize networks, and apply various statistical models to analyze network structures and related outcomes. The program will generate an interactive report that integrates findings from descriptive statistics and statistical models, presenting results in both numerical/table format and empirical interpretations with practical implications.

  • Develop a desktop application capable of handling social network data for analysis.
  • Implement features to collect, upload, process, and validate network data within the application.
  • Design and integrate various descriptive statistics algorithms to capture key network metrics.
  • Develop a network visualization module to enable users to explore and understand network structures.
  • Incorporate statistical models such as ERGMs or ALAAMs for analyzing network properties.
  • Enable users to customize and apply statistical models to investigate specific network outcomes.
  • Generate an interactive report that presents the results in a user-friendly and informative manner.
  • Provide users with interpretive insights and practical implications based on the analysis results.
